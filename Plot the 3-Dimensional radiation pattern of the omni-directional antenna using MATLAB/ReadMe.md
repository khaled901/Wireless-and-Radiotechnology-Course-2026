# 3D Radiation Pattern of Omni-directional Antenna

This project uses MATLAB to plot the 3D radiation pattern of an omni-directional antenna.

## Description

The radiation pattern is generated using spherical coordinates and converted to Cartesian coordinates using the sph2cart function. Since the field pattern is constant (E = 1), the output is a spherical shape representing uniform radiation in all directions.

## Input Parameters

Theta range: 0 to 360 degrees
Phi range: 0 to 360 degrees
Field pattern: E(THETA,PHI) = 1

## Output

A 3D spherical radiation pattern plotted using MATLAB.
